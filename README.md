# 🐱🚗 Котик за кермом — Оболонь

Веб-гра: катайся **котиком-машинкою реальними вулицями Оболоні** (Київ) і об'їдь усі
8 локацій якнайшвидше. На кожній — маленька підказка з ПДР України.

**▶ Грати:** відкрий сторінку GitHub Pages цього репозиторію у будь-якому браузері
(працює і на iPhone, і на Android, і на комп'ютері — нічого встановлювати не треба).

## Керування
- 📱 **Телефон:** кнопки внизу — ◀ ▶ повороти, **Газ**, **Гальмо**.
- 💻 **Комп'ютер:** стрілки `← ↑ → ↓` (або `W A S D`).

## Локації Оболоні
Метро «Оболонь», «Мінська», «Героїв Дніпра», ТРЦ Dream Town, озеро Опечень,
Оболонська набережна, парк «Наталка», Оболонський проспект.

## Як це зроблено
Один файл `index.html` — HTML + CSS + JavaScript. Карта — [Leaflet](https://leafletjs.com/)
з тайлами CARTO/OpenStreetMap. Жодного бекенду, жодної збірки — чиста статика,
тому й хоститься безкоштовно на GitHub Pages.

## Запустити локально
Просто відкрий `index.html` у браузері (потрібен інтернет для завантаження карти).
